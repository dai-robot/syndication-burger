# Syndication Burger! — アートディレクション

## ビジュアル方向性

- **ポップな2D** — 丸み、グラデーション、影で立体感
- **明るいキッチン** — 暖色背景（#FFF8F0）、タイル模様、柔らかい照明
- **おいしそうな具材** — 単色円/四角禁止。ハイライト・影・テクスチャ風ストローク
- **UI** — 角丸カード、パステルアクセント、デフォルトHTMLボタン不使用

## 具材描画方針

各具材は Phaser Graphics で以下を含む:

- **グラデーション風** — 複数レイヤーの楕円/パス
- **ハイライト** — 左上に白半透明楕円
- **影** — 右下に暗色オフセット
- **特徴的シルエット** — レタス波、トマト輪切り、チーズ角、パティ焼き目

## 必須演出

| 演出 | 実装 |
|------|------|
| 着地揺れ | scale/sin オシレーション 300ms |
| コンボキラキラ | パーティクル + テキストポップ |
| ケチャップ | ベジェ曲線 + 滴パーティクル |
| 完成圧縮 | バーガー全体 scaleY 0.85→1.0 イージング |
| 星評価 | 順次スケールイン + バウンス |
| 落下ガイド | 中央縦線 alpha 0.15 |

## 禁止事項

- 絵文字のみの具材表現
- 単色の四角/円だけ
- スカスカな画面（背景装飾必須）
- 安っぽいデフォルトUI

## タイポグラフィ

- タイトル: bold, 32px+, オレンジグラデーション風
- HUD: 14〜18px, 読みやすいコントラスト
- コンボ: 24〜36px, ポップカラー（Nice=#4ECDC4, Great=#FFD93D, Perfect=#FF6B35）
