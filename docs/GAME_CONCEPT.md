# Syndication Burger! — ゲームコンセプト

## 仮タイトル

**Syndication Burger!**

## 一言コンセプト

下バンズの上に具材が次々と落ちてくる。トレイを左右に動かして具材を積み、ケチャップをかけ、上バンズで閉じて完成させる、30〜90秒のカジュアル積み上げゲーム。

## コア体験

1. **料理の気持ちよさ** — 具材がポスッ、ドスッ、ぺたっと積まれる手触り
2. **積みゲーのハラハラ** — 少しズレたり崩れそうになる緊張感
3. **完成の達成感** — 上バンズで閉じ、星評価とスコアで褒められる

## ターゲット

- スマホで短時間（1プレイ30〜90秒）遊べるカジュアルプレイヤー
- 子どもから大人まで、説明なしで直感的に遊べる難易度
- リトライしたくなる「もう一回だけ」のループ

## MVPの範囲（プロデューサー判断）

### 含める

- スタート画面 / 結果画面
- 7種具材 + 上下バンズ + ケチャップフェーズ
- 注文カードと一致ボーナス
- センター精度・安定度・コンボ・ソース評価によるスコア
- 星評価（1〜3星）
- Web Audio API による BGM / 効果音
- スマホ縦画面・片手操作（ドラッグ + キーボード）

### 含めない（将来拡張）

- オンラインランキング
- 複数ステージ / キャンペーン
- 外部画像・音源アセット
- 課金・広告
- マルチプレイ

## 成功基準

- 初見30秒以内に操作を理解できる
- 完成バーガーを見て「もう一回」と思える
- 具材ごとに見た目・挙動・音が異なる
- npm install → npm run dev で即プレイ可能
