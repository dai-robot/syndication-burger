# Syndication Burger! — ゲーム仕様書

## 画面構成（縦画面 390×844 基準）

| 領域 | 内容 |
|------|------|
| 上部 HUD | 注文カード、スコア、コンボ、残り具材数、BGMトグル |
| 中央 | 落下プレイエリア（明るいキッチン背景、薄い落下ガイド） |
| 下部 | トレイ、下バンズ、操作ガイド |
| 完成時 | 完成バーガー、星評価、スコア内訳、リトライボタン |

## 操作

| 入力 | 動作 |
|------|------|
| ドラッグ（タッチ/マウス） | トレイ左右移動 |
| ← → キー | トレイ左右移動（PC） |
| ケチャップフェーズ長押し | ソース量調整 |
| BGMボタン | 音 ON/OFF |

## コアゲームループ

1. スタート画面
2. 注文カード表示
3. 下バンズ配置
4. 具材順次落下 → プレイヤーが受ける
5. ケチャップフェーズ（長押し）
6. 上バンズ落下
7. スコア計算・星評価
8. リトライ

## 具材仕様

| 具材 | 特性 | 着地演出 |
|------|------|----------|
| 下バンズ | 最初配置、弾力あり | 軽いバウンス |
| パティ | 重い、安定 | ドスッ（低い音） |
| チーズ | 薄く広がる | ぺたっ + 垂れ |
| レタス | 軽い、波打ち | ふわっ揺れ |
| トマト | 滑りやすい | 弾む、中心外れ減点 |
| ピクルス | 小さい、ボーナス | かわいいズレ許容 |
| ケチャップ | ソースフェーズ | にゅるっ |
| 上バンズ | 最後、完成 | ぎゅっ圧縮 |

## スコア要素

| 要素 | 配点イメージ | 条件 |
|------|-------------|------|
| センター精度 | 0〜200/具 | 中心からの距離 |
| 注文一致 | +150 / -50 | 注文通り / 不要具材 |
| 安定度 | 0〜100/具 | 累積ズレ量 |
| ソース評価 | -50〜+150 | 適量 / 過不足 |
| コンボ | ×1.0〜×2.0 | 連続センター配置 |
| 完成ボーナス | 0〜500 | 上バンズ精度 |

**星評価:** 総合スコア率 — ★1: 40%+, ★2: 65%+, ★3: 85%+

## 失敗条件（MVP: 減点方式）

- 具材が画面外落下 → ミス（-100点）
- 3回ミスで追加ペナルティ（-200点）
- ゲームオーバーなし（子ども向け難易度）

## 難易度曲線

| ラウンド | 変化 |
|----------|------|
| 1 | 低速、具材5種、揺れなし |
| 2+ | 落下速度↑、横揺れ↑、具材数↑、ソース判定厳格化 |

## ゲーム状態

```
INTRO → STACKING → KETCHUP → TOP_BUN → COMPLETE → RESULT
```
